# 🌐 Lyoq Labs  
Lyoq Labs is an unofficial creative project focused on modern, animated, black & white profile pages.  
All profiles are hosted via GitHub Pages:

➡️ **https://lyoqlabs.github.io/USERNAME**

---

## ⚡ What We Do
We create:

- social links profile for users 
- downloads such as Minecraft texture packs, and more 
- downloads such as Roblox scripts & executors (not created by us).
> [!CAUTION]
>  Read our [README on our Roblox repository](https://github.com/lyoqLabs/roblox/blob/main/README.md) for more informations. 

---

## 🔍 Upcoming Features
### **User Search System**
A username search is in development.  
If a profile doesn't exist, the system will respond with:

- **Available**  
- **Not Found**  
- **Banned**  
- **For Sale**
